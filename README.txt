
		小兵以太网测试仪简介
		
======================

    小兵以太网测试仪(下文称"本软件")是一款基于pc机网卡实现的测试仪工具。 
    主要功能：构造编辑报文流、发包、抓包。
    
    本软件是一款开源、免费软件。
    具体版权说明见COPYING.txt。

    本软件的编译方法见Build.txt。

======================

2013年10月，1.0发布
2013年11月，2.0.0发布
2013年11月，2.3.0发布，同时软件走向开源
